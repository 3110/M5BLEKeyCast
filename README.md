# M5BLEKeyCast

`M5BLEKeyCast`は複数の M5Stackデバイス（ATOM Matrix/Lite）を BLE キーボードにして，`M5BLEKeyCast`が動いているすべてのM5Stackデバイスが同時に同じキーを入力できるようにします。

もともと，`M5BLEKeyCast`はロボホンを同期させて動かすために作ったものです。例えば，ロボホン側はロブリックでキー入力を待ち，ダンスをさせるスクリプトを動かしておきます。各ロボホンにBLEキーボードとしてM5Stackをそれぞれ接続しておけば，ボタンを押すことでロボホンを同期させて踊らせることができます。

## 実行方法

`M5BLEKeyCast`を実行しているM5Stackデバイスをキー入力したいデバイスにBLEキーボードとしてペアリングします。

ペアリングするとM5StackデバイスのLEDが青点灯します。M5Stackデバイスのボタンを押すと，そのデバイスに記録されたキーを ESP-NOW でブロードキャストし，全M5Stackデバイスが一斉に同じキーを送信します。

LEDの色が表す状態は以下の通りです。

| 色     | 状態                     |
|--------|--------------------------|
| 消灯   | BLE 未接続               |
| 青点灯 | BLE 接続済み             |
| 緑点滅 | ESP-NOW 受信 → キー送信 |
| 黄点灯 | 設定モード               |

### 設定モード

ボタンを押しながらM5Stackデバイスを起動すると，LEDが黄点灯して設定モードに入ります。

シリアルコンソールに以下のメニューが表示されます。

```
=== Config Mode ===
[1] Set key  (current: '1')
[2] Set name (current: 'KC_Lite')
[0] Exit
Enter choice:
```

| 選択 | 設定内容 |
|------|----------|
| `1`  | 送信キーを変更する。印字可能なASCII文字を1文字入力してください |
| `2`  | BLEデバイス名を変更する。最大20文字の名前を入力してEnterを押してください |
| `0`  | 設定モードを終了して再起動する |

設定はその場でNVSに保存されます。`0` で終了して再起動するまで繰り返し設定を変更できます。デバイス名を変更した場合は，ペアリング済みのデバイスで再ペアリングが必要になることがあります。
